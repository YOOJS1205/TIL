# 버블 정렬

- ## 왜 정렬?

  이진 탐색을 사용하기 위해서는 정렬된 상태여야 하는 가정이 필요하다.

- ## 정의

  두 개의 인접한 자료 값을 비교하면서 위치를 교환하는 방식으로 정렬

- ## 예시

  [3, 2, 4, 6, 5, 7, 8, 1]<br>
  **1**: 3 vs 2, 4, 6, 5, 7, 8, 1<br>
  [1, 2, 4, 6, 5, 7, 8, 3]<br>
  **2**: 2 vs 4, 6, 5, 7, 8, 3
  ...<br>
  가장 큰 값이 오른쪽에 점점 쌓인다.

- ## BIG O

  (n-1) \* (n-1) => O(n^2)

- ## Omega
  이미 정렬된 상태에도 정렬을 진행한다. => n^2
